CONSTRAINTS    |   Rules Your Agent Must Follow 
Chain 
Must make at least 3 tool calls in a single reasoning chain. E.g., 
lookup user → check order → send reply. 
Recover 
Must gracefully handle tool failures - at least one tool will timeout or 
return malformed data. Don't crash. 
Tickets must be processed concurrently, not one-by-one. Sequential 
processing will be penalised. 
Every decision must be explainable. No black-box outputs. 
AI coding tools are fully allowed - but you will be asked to walk through your code live. If you can't explain it, 
it counts against you. Recover 
Concurrency Explain
